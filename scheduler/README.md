# Scheduler Library (Private)

This directory contains the proprietary `scheduler` library which is not publicly available due to copyright restrictions.

## For Reviewers

If you are a reviewer and need access to run the code, please contact the authors for access credentials.

## Structure

The `scheduler` library provides:
- `scheduler.rl_model` - Reinforcement learning models and training
- `scheduler.dataset_generator` - DAG dataset generation
- `scheduler.tools` - Utility tools for evaluation and analysis

## Installation

This library is included as a private dependency. Contact the authors for access.
